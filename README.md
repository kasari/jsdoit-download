# jsdoit-download
Download jsdo.it user's code tool

## Install
```
npm install -g jsdoit-download
```

## Usage
`jsdoit-download -h`
```
Usage: jsdoit-download [options]

Options:
  -u, --user <name>  user name
  -d, --dir <path>   output dir (default: ".")
  -h, --help         output usage information
```

## Example
```
jsdoit-download -u amusan
```
